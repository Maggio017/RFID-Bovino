Monitoramento Bovino com RFID e Arduino

Projeto de leitura de tags RFID para identificar animais, monitorar tempo de alimentação/consumo e gerar dados para análise no agronegócio.

Ver Projeto

Resumo

Aplicação prática de um sistema embarcado (Arduino/ESP32) que usa leitores RFID RC522 para identificar bovinos por tags passivas, centralizar leituras e gerar logs para análise de comportamento de alimentação e bem-estar.

Benefícios

  • Rastreamento automático da presença em bebedouros e comedouros.
  • Coleta de dados para análises (tempo de permanência, padrão de consumo).
  • Baixo custo de implementação e fácil manutenção.

Tecnologias

  • Hardware: Arduino / ESP32, RC522 (RFID), sensores, relés.
  • Software: C/C++ (Arduino IDE), comunicação serial, logs CSV/JSON.
  • Dados: Tratamento e visualização com ferramentas como Excel / Power BI ou scripts Python.
